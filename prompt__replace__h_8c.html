t_err ft_retrieve_hostname ( char **  replacement)

Retrieve hostname and save in provided pointer.

The current hostname is saved in /proc/sys/kernel/hostname. Open this file and retrieve hostname via get_next_line(). Hostname is searched for the first '.' which is replaced by '\0'. If no '.' is found hostname is delimited by newline: Search for '
' and replace with '\0' Last get_next_line() call to empty buffer.

Parameters
replacementPointer pointer where to save string.
Returns
t_err SUCCESS, ERR_OPEN, ERR_MALLOC
