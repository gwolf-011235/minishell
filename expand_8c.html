Detailed Description

Functions to exec different expansions depending on token type.

Contains driver function to handle expansion and quote removal.

t_hashtable *  symtab

Expands ARG token.

Init tracker with ft_init_tracker(). Expand and quote removal with special ft_expander_arg(). If expansion on unquoted var: check with ft_field_split() if node has to be split. If ft_field_split() fires it resets the tracker to the last node of the split to continue expansion. If expanded node is empty, and was no quotes found mark it as DELETE.
