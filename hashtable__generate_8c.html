Creates and initializes a new t_hashtable.

Mallocs the struct, then sets size to param size and hashfunction to param hf. Then enough pointers get calloc'd to fit size. If malloc of hashtable or calloc of elements fail NULL gets returned.

Parameters
sizeSize of hashtable - that many elements pointer get alloc'd
hfPointer to hashfunction which should be used
Returns
t_hashtable* pointer to malloc'd hashtable; NULL if malloc fail
