t_err ft_create_env_pwd ( char **  pwd,
t_buf *  buf 
)

Creates a new env_var PWD=path/to/pwd.

Receives pointer to pointer to change value outside of function. Return is t_err, so it can't return pointer itself. Call ft_create_pwd_value to get right side of env_var. If error passes on the error. If success strjoin with left side.

Parameters
pwdPointer to pointer where to save string.
bufPointer to buffer.
Returns
t_err SUCCESS, ERR_MALLOC, ERR_CWD_FAIL

◆ ft_create_pwd_value()

t_err ft_create_pwd_value ( t_buf *  buf)

Create string with the present working directory.

Receives pointer to pointer to change value outside of function. Return is t_err, so it can't return pointer itself. Malloc string with 4096 and call getcwd for pwd. If path is longer than size, free and malloc twice the size. Repeat until path fits in array.

Parameters
bufPointer to buffer.
Returns
t_err SUCCESS, ERR_MALLOC, ERR_CWD_FAIL.
