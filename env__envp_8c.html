Create a NULL-terminated environment pointer (envp)

Malloc char array of num_values of hashtable plus 1 for terminator. Fetch all elements with ft_envp_fill() with switch set to false. Handles errors.

Loop through all elements and copy the pointers.

The hashtable has an elements array, which are all linked lists. The function has to loop trhough the whole array. And per array it has to loop through the linked list. This function only copies the pointers in the hashtable. Only variables which have a value are copied.

Parameters
htEnvironment table which to copy.
envpWhere to save pointers.
Returns
t_err SUCCESS
