Expand an environment variable in string.

Check if special variable - handle with ft_special_var(). Else search for variable name and get the variable value. Increase token.len to include $. Insert the replacement in the string. Update the position to be after the inserted replacement.

Parameters
inputInput string.
symtabEnvironment.
posCurrent position.
infoStruct containing ret_code and shell_name.
Returns
t_err SUCCESS, ERR_NOEXPAND, ERR_MALLOC.
