void ft_exit ( char **  argv,
bool *  loop 
)

Exit the current shell instance.

Check size of given args. If at least one arg is given check if it is a number with ft_is_number(). If not a number print error msg and set return code but continue. If a exactly one number arg convert it with ft_atoi(). If arg1 is a number but more than one arg print error msg and return(!). Free t_hashtable. Call exit() with error code.

argvNULL terminated args.
