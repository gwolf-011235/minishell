t_err ft_export ( char **  argv,
t_hashtable *  env_tab 
)

Export a variable into environment.

Check arg size with ft_get_array_size(). If no arg is given ft_print_env_sorted(). Save_err is set to SUCCESS. For all args try to insert them into environment with ft_check_and_update_env(). If error is encountered print error with ft_export_error() and save error code. Continue for all other args. At the end return saved error

Parameters
argvNULL terminated args.
env_tabEnvironment.
Returns
t_err SUCCESS, ERR_EMPTY, ERR_INVALID_NAME
