◆ ft_handle_input()

void ft_handle_input ( char *  input,
t_data *  data 
)

Driver function process input.

Input is checked for syntax errors. Input is lexed, splitted into tokens which are saved in a token list. Token list is iterated and expanded if necessary. The expanded token list is parsed and the content categorised into a list of cmds. The list of cmds is processed by the executor.

Parameters
inputInput string.
dataOverarching data struct containing the env.
Returns
t_err ERR_EMPTY, ERR_MALLOC, ERR_SYNTAX, ERR_PIPE, SUCCESS

Create a prompt string which gets expanded.

Search for provided var $PS1 or $PS2 in sym_tab. If not found set prompt to provided standard. If found ft_strdup() value of var. Expand the string with ft_expand_prompt_str().
