t_err ft_create_shlvl_env_str ( char **  shlvl)

Create an env_var SHLVL starting at 1.

If there is no SHLVL present, this function creates one. It always starts at 1.

Parameters

t_error SUCCESS or ERR_MALLOC

◆ ft_increment_shlvl()

t_err ft_increment_shlvl ( t_data *  data)

Increments the env_var SHLVL.

Search for SHLVL, if not found return error. Extract the number of the found SHLVL and add 1 to it. If the result is bigger than MAX_SHLVL return error. Create a new env_string by converting number to string and join. Update the env_var node by freeing the old string and updating pointers.

Parameters
dataBig data struct.
Returns
t_error SUCCESS, ERR_NO_SHLVL, ERR_MAX_SHLVL, ERR_MALLOC.
