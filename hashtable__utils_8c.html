char *  string,
size_t  keylen

Delete one element of hashtable.

Check if params are empty. Generate index and set tmp pointer to start of list. Set prev pointer to NULL. Loop through the list and update tmp and prev pointers. If tmp == NULL no element was found. If prev == NULL element is at start of list. If element is found free. Then free element and update next pointer.

Parameters
htHashtable in which to delete one element
stringenv_string which should be deleted
keylenlength of env_var (everthing before =)
Returns
t_err SUCCESS, ERR_EMPTY, ERR_HT_NO_DELETE
