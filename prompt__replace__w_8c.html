Tries to replace env_var $HOME in string with tilde.

Search for $HOME. If found ft_strncmp with string: it should only replace if string starts with $HOME. Replace the token $HOME with '~' with ft_replace_token(). Also returns SUCCESS if $HOME not found or ft_strncmp not zero - no error.

Parameters
replacementWhere to replace $HOME with tilde.
sym_tabEnvironment.
Returns
t_error SUCCESS, ERR_EMPTY, ERR_MALLOC

Creates replacement string for token \w.

Search for env_var $PWD. If found ft_strdup value and save in replacement. If not found create UNKNOWN with ft_replace_not_found(). Try to replace $HOME with ft_replace_tilde().

Parameters
replacementPointer pointer where to save string.
