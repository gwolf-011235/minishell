Calculates the len of the string, where tokens have been replaced.

Check if either of the provided pointers is empty. Calc the strlen of token and replacement. Search for instances of token in string. If one is found add len upto token, add replacement len, then jump over token. Search again. If no token is found add remaining len of str.

Parameters
strThe string for which the expanded len should be calculated.
tokenChar string which is going to be replaced.
replacementChar string as replacement for token.
expanded_lenPointer to size_t in which to save the len.S

Copies str into expanded_str while replacing each token.

Check if either of the pointers is empty. Search for token in str. If token is found, everything upto token is copied. Then replacement is copied and pos jumps over token. If no token is found copy the remaining str.

Parameters
expanded_strPointer pointer to buffer wherein to copy.
strPointer to str which gets copied and expanded.
tokenChar string which should be replaced.
replacementChar string which is inserted instead of token.

This function finds token in str and replaces it.

Calculate expanded len of str. Malloc a buffer which holds expanded len and zero terminate. Copy str into buffer whereby each token is replaced. Free str and update pointer to the expanded string.

Parameters
strPointer pointer to char string which is expanded.
tokenChar string to search for.
replacementChar string which replaces token.
Returns
t_err SUCCESS, ERR_MALLOC, ERR_EMPTY
