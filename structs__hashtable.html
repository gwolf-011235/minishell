Detailed Description

Hashtable with external chaining.

This hashtable is used to store the env_vars. It has <size> * buckets in which to save elements. Hash function is used to generate index from the env_var name (e.g. PWD). The env_var string is then saved in a node at the index. If the index is already used external chaining is used. The new node is inserted at the front of this linked list.
