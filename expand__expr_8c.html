Handle double quotes.

Remove found double quote with ft_eat_char(). Switch bool in_quotes on/off. This way we know if we are in double quotes or not. The next time we see a double quote the switch gets flipped again.

Parameters
exprString.
posCurrent position.
in_double_quotesPointer to change switch.
Returns
t_err SUCCESS.
