t_err ft_create_env_shlvl ( char **  shlvl,
int  val 
)

Create an env_var SHLVL starting at 1.

If there is no SHLVL present, this function creates one. if val is 1 just ft_strdup() the str and return. Else convert val to char* with ft_itoa(). Create new str with ft_strjoin().

Parameters
shlvlPointer pointer to change value outside of function.
Returns
t_error SUCCESS, ERR_MALLOC

◆ ft_increment_shlvl()

t_err ft_increment_shlvl ( t_hashtable *  env_table)

Increments the env_var SHLVL.

Search for $SHLVL, if not found return error. Extract number of found $SHLVL and add 1 to it. If result is higher than MAX_SHLVL return error. Create a new env_string with ft_increment_shlvl(). Update $SHLVL with ft_hashtable_swap().

Parameters
env_tableEnvironment.
Returns
t_error SUCCESS, ERR_NO_SHLVL, ERR_MALLOC, ERR_EMPTY, ERR_HT_NO_SWAP.
