bool ft_is_env_assign ( char *  str)

Checks if string is a valid environment assignment.

First char has to be alphabetical or underscore. Following chars can also include numbers. Char after the variable key has to be '='.

Parameters
strThe string to check
Returns
true Valid assignment.
false Not an assignment.
